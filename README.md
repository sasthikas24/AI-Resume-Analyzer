# AI Resume Analyzer

## Overview
AI Resume Analyzer is a web-based application that extracts skills and key information from resumes using Natural Language Processing techniques.

## Tech Stack
- Python
- Flask
- Natural Language Processing (NLP)
- spaCy
- HTML, CSS

## Features
- Resume parsing
- Skill extraction
- Keyword matching
- Simple UI

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the Flask application

## Future Enhancements
- Resume scoring
- Job description matching
- Dashboard analytics
